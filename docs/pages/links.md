# 友情链接

友情链接页面展示友站信息，需要安装 [链接管理插件](https://www.halo.run/store/apps/app-links)。

## 展示样式

主题提供三种友链展示样式，在 [友情链接配置](/config/links) 中设置：

### Cards（卡片）

默认样式，每个友链以卡片形式展示，包含头像、名称和描述。视觉效果丰富。

### Grid（网格）

紧凑的网格布局，以图标和名称为主，适合友链数量较多的站点。

### List（列表）

列表布局，每行一个友链，信息密度适中。

## 友链管理

友情链接通过 Halo 后台的 **链接** 管理功能添加：

1. 进入 Halo 后台 → **链接** → **新建**
2. 填写链接名称、URL、描述和头像
3. 可将链接分组，主题会按分组展示
